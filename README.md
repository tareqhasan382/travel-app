## ğŸš€ Getting Started

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/tareqhasan382/travel-app.git
   cd react-native-travel-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Install iOS dependencies** (macOS only)
   ```bash
   cd ios && pod install && cd ..
   ```

### Running the App

#### For iOS
```bash
npm run ios
# or
yarn ios
```

#### For Android
```bash
npm run android
# or
yarn android
```

#### Start Metro Bundler
```bash
npm start
# or
yarn start
```
